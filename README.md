# Zenn Contents

* [📘 How to use](https://zenn.dev/zenn/articles/zenn-cli-guide)

  * 🐳 dockerを起動する

    ```$ docker compose up -d```

  * 🐳 dockerをやめる

    ```$ docker compose down```

  * 👇 新しい記事を作成する

    ```$ zenn new:article```

  * 👇 新しい本を作成する

    ```$ zenn new:book```

  * 👇 表示をプレビューする

    ```$ zenn preview```
